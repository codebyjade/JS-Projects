## ☕ 1. Cafe Finder — Google Maps

Find cafes near you on an interactive map with ratings, distance and open/closed status.

**Features:**
- Search cafes by city or neighborhood
- 📍 Detect your location automatically
- Shows ratings, open/closed, distance
- Click cafe card to pan the map

**Setup:**
1. Get free API key → [Google Cloud Console](https://console.cloud.google.com)
2. Enable **Maps JavaScript API** + **Places API**
3. Replace `YOUR_API_KEY` in the `<script>` tag
4. Open in browser ✅

## 🎙️ 2. Voice Virtual Assistant — ElevenLabs

A talking AI assistant with a stunning dark orb UI. Speak or type — it responds with a real human voice.

**Features:**
- 🎤 Voice recognition via Web Speech API
- 🔊 Human-like voice via ElevenLabs TTS
- Free Browser TTS fallback — no API needed
- Multiple voice options
- Full conversation history

**Setup:**
1. Get free API key → [ElevenLabs](https://elevenlabs.io)
2. Replace `YOUR_ELEVENLABS_API_KEY` in the JS
3. Or just select **"Browser TTS"** — works for free ✅

## 🌭 3. Not Hotdog Detector — Hugging Face

Upload any food image and AI tells you — hotdog or not hotdog? Inspired by Silicon Valley 😄

**Features:**
- 📸 Upload image or drag & drop
- 🔗 Paste image URL to analyze
- Confidence score + top prediction labels
- Demo mode works without API key

**Setup:**
1. Get free token → [Hugging Face](https://huggingface.co/settings/tokens)
2. Replace `YOUR_HF_API_KEY` in the JS
3. Open in browser and upload a food pic ✅
